PELSA-Decipher is a freely available software tool aimed at processing quantitative datasets acquired from PELSA experiments. PELSA-Decipher enables simultaneous ligand-binding protein identification, binding region localization, and binding affinity calculation.

PELSA-Decipher is written in python and R language, with a graphical user interface (GUI) written in PyQt5. Instructions on software installation, operation, common problems, and other issues could be found in User manual in “PELSA-Decipher-Beta01” folder.

R scripts can generate the same analysis results as PELSA-Decipher (except that they do not generate local stability plots). R scripts are provided so that users can modify the scripts to customize the output files based on their needs.
